Instructions

How to complete the text box:

  • Follow the prompt in each text box located on the second page of the profile.
  • Please list two separate examples for each indicator.
  • Please check your spelling and grammar before printing.
  • When done listing examples, read on to save the file.

How to save the file:

  • Click the print Icon located at the top of each page
  • Follow the prompts to ensure the text entered is correct.
  • After following the prompts, click the menu button in the top right and then click print, or press Ctrl + P.
  • In the print preview box that opens, click on Page Setup...
  • In the Format & Options section, check Print Background.
  • In the Margins & Header/Footer section, put all margins as 0.5 and select --blank-- for all of the dropdowns under Headers & Footers.
  • Now click Print..., change the Printer Name to Microsoft Print to PDF and click OK.

Instructions

How to complete the text box:

  • Follow the prompt in each text box located on the second page of the profile.
  • Please list two separate examples for each indicator.
  • Please check your spelling and grammar before printing.
  • When done listing examples, read on to save the file.

How to save the file:

  • Click the print Icon located at the top of each page
  • Follow the prompts to ensure the text entered is correct.
  • After following the prompts, click the settings button in the top right and then click Page Setup (under the print command).
  • In the Paper Options section, check Print Background Colors and Images and Enable Shrink-to-fit
  • Select -Empty- for all of the dropdowns under Headers & Footers.
  • Now exit the Page Setup and go to Print Preview.
  • Ensure that Shrink To Fit is selected in the dropdown menu
  • Now click the print icon, change the Printer Name to Microsoft Print to PDF and click OK.
